# django-rest-SAP
## djangorestframework 라이버르러리를 이용해 restAPI 구축

폴더설치후

pip install django
pip install djangorestframework
pip install Pillow(필요시)

manage.py 있는 위치에서

python manage.py runserver 0:8000 (도커로 실행했다면 0:8888 식으로 모든 포트를 열어줘야한다)
